## 快速开始

### 访问系统

**PC端访问：**
1. 打开浏览器（推荐：Chrome、Firefox、Safari、Edge）
2. 在地址栏输入系统网址：`http://your-domain.com` 或 `http://localhost:8080`（本地开发）
3. 页面跳转至登录页面

**登录步骤：**
1. 输入**用户ID**（学号或工号）
2. 输入**密码**
3. 点击**"登录"**按钮
4. 登录成功后自动跳转至首页

### 首次登录

如果您是首次登录，请注意：
- **默认密码**：由管理员在创建账户时设置
- **修改密码**：建议首次登录后立即修改密码
  - 进入"个人设置" → "修改密码"
  - 输入原密码和新密码，确认保存

### 3.3 在线考试

**访问路径：** 课程详情→考试列表

#### 3.3.1 查看可参加考试

**页面显示：**
| 考试名称 | 出题教师 | 状态 | 分数 | 操作 |
|---------|--------|------|------|------|
| 第1章测试 | 李老师 | 已考 | 85 | 查看 |
| 第2章测试 | 李老师 | 进行中 | - | 进入 |
| 期末考试 | 李老师 | 未开始 | - | - |

**状态说明：**
- **未开始**：考试时间还未到达，禁止进入
- **进行中**：可以进入考试，继续答题或重新进入答题
- **已考**：已参加并提交，可查看成绩和试卷
- **已结束**：考试时间已过，可查看成绩

#### 3.3.2 进入在线考试

**重要提示：**
- ⚠️ 建议使用**稳定的网络**进入考试
- ⚠️ 使用**台式电脑或笔记本**，避免使用手机
- ⚠️ 提前**10分钟**进入考试做准备

**进入步骤：**
1. 在考试列表中，找到状态为"进行中"的考试
2. 点击**"进入考试"**按钮
3. 系统弹出确认对话框，点击**"确认进入"**
4. 系统生成个人的随机试卷并显示在屏幕上

#### 3.3.3 答题界面

```
┌─────────────────────────────────────────────────────┐
│  Java高级 - 第2章测试                               │
│  剩余时间: 00:45:30 ⏱️                               │

│                                   │                │

└─────────────────────────────────────────────────────┘
```

**答题技巧：**

| 题型 | 操作方法 | 说明 |
|------|--------|------|
| 单选题 | 点击选项前的圆圈 | 单选，不可多选 |
| 多选题 | 点击选项前的方框 | 可多选，需全选才对 |
| 判断题 | 选择"正确"或"错误" | 二选一 |
| 填空题 | 在文本框内输入答案 | 支持空格和符号 |
| 简答题 | 在文本框内输入答案 | 自由作答，教师批改 |

**自动保存：**
- 系统每30秒自动保存一次答案
- 接近提交时会有提醒
- 即使网络中断，已保存的答案不会丢失

**提交试卷：**
1. 答题完成或时间将到时，点击**"提交试卷"**按钮
2. 系统显示确认对话框
3. 再次确认点击**"确认提交"**
4. 试卷提交成功，显示初步成绩

**重要说明：**
- ⚠️ 提交后**不能修改答案**
- ⚠️ 考试**时间到自动收卷**
- ⚠️ 客观题立即自动评分，主观题等待教师批改

#### 3.3.4 查看考试成绩

**步骤：**
1. 在考试列表中，找到已考过的考试
2. 点击**"查看"**按钮
3. 进入成绩详情页面

**成绩详情包含：**
- 试卷名称和考试时间
- 最终得分和班级平均分
- 答题耗时
- 单题解析：每道题的答案、得分、教师评语

**客观题显示：**
- 题目内容
- 你的答案
- 标准答案
- 得分（自动评分）

**主观题显示：**
- 题目内容
- 你的答案
- 教师给分
- 教师评语/反馈

```
   答题区：[多行文本输入框]

```

#### 3.4.3 保存与提交

**保存草稿：**
1. 答题过程中随时点击**"保存草稿"**
2. 系统保存当前答案，不影响后续修改
3. 提交前可多次保存

**提交作业：**
1. 答题完成后，点击**"提交作业"**
2. 系统显示确认对话框
3. 确认无误后点击**"确认提交"**
4. 作业提交成功，进入"已提交"状态

**修改已提交作业：**
- 在截止时间前，可点击**"重新提交"**再次修改
- 系统会记录所有提交版本
- 教师批改的是最后一次提交的版本

#### 3.4.4 查看作业评分

**步骤：**
1. 在作业列表中，找到已批改的作业
2. 点击**"查看评分"**按钮
3. 进入评分详情页面

**显示信息：**
- 最终得分和分数分布
- 每道题的得分
- 教师的整体评语
- 改进建议

## 常见问题解答

### 6.1 登录与账户相关

**Q：忘记密码怎么办？**
A：
1. 在登录页面点击**"忘记密码"**链接
2. 输入您的用户ID（学号或工号）
3. 系统发送重置链接到您的邮箱
4. 点击邮件中的链接重置密码
5. 如无法收到邮件，请联系管理员

**Q：账户被禁用，无法登录？**
A：该账户已被管理员禁用。请：
1. 确认自己的账户状态
2. 联系系统管理员申请启用
3. 可能原因：违反使用规则、学期结束等

**Q：同一个账户可以在多个地方同时登录吗？**
A：支持。一个账户可以同时在多个浏览器或设备上登录，但建议在不使用时及时登出。

**Q：修改密码后还是无法登录？**
A：
1. 确保Caps Lock(大小写切换键)未被激活
2. 确认密码中没有多余空格
3. 清除浏览器缓存后重试
4. 如仍无法解决，联系技术支持

### 6.2 课程与学习相关

**Q：如何查看已学课程的学习进度？**
A：
1. 进入课程详情页面
2. 在左侧菜单选择**"学习统计"**
3. 显示详细的学习进度、时长等数据

**Q：可以重复学习同一个课程吗？**
A：可以。完成课程后仍可继续访问，复习相关内容。

**Q：课程资源无法下载，怎么办？**
A：
1. 检查网络连接是否正常
2. 确认浏览器是否允许下载
3. 尝试其他浏览器
4. 文件可能已被删除或移除，联系教师

**Q：视频加载很慢或无法播放？**
A：
1. 检查网络速度（建议4Mbps+）
2. 切换到标清或流量节省模式
3. 清除浏览器缓存
4. 换个浏览器或设备尝试
5. 可尝试下载后离线观看

### 6.3 考试相关

**Q：进入考试后网络断了怎么办？**
A：
1. 不用担心，已回答的题目已自动保存
2. 重新连接网络后，刷新页面继续答题
3. 如自动保存失败，系统会在重新连接时恢复答题状态

**Q：考试时间到了但我还没做完，怎么办？**
A：
1. 考试到点后系统会自动收卷
2. 已保存的答案会被提交评分
3. 未保存的答案将丢失
4. 建议在考试开始前确保计算机稳定

**Q：如果误操作提交了怎么办？**
A：
1. 提交后无法修改答案
2. 可重新参加考试（如教师开启了重考）
3. 建议认真检查后再提交

**Q：如何查看我的考试成绩？**
A：
1. 进入课程→考试列表
2. 找到已考的考试，点击**"查看"**
3. 显示成绩和详细解析

**Q：主观题为什么还没有分数？**
A：主观题需要教师手工批改，一般会在考试结束后3-7天内完成。您可以进入考试详情页面查看批改进度。

### 6.4 作业相关

**Q：作业超过截止时间仍可提交吗？**
A：
1. 可以继续提交，但会标记为**"逾期"**
2. 教师仍会进行批改评分
3. 但可能会影响最终成绩、扣分等

**Q：如何修改已提交的作业？**
A：
1. 进入作业页面，点击**"重新提交"**
2. 修改答案后再次提交
3. 系统会保留所有提交版本，教师批改最后一次提交

**Q：作业成绩查询不到？**
A：
1. 作业刚提交不是立即评分，需要教师批改
2. 一般3-5天内会得到评分
3. 可在"批改进度"查看当前状态
4. 如超期未批改，可联系教师

### 6.5 技术问题

**Q：系统支持哪些浏览器？**
A：
- Chrome 90+ ✓推荐
- Firefox 88+ ✓
- Safari 14+ ✓
- Edge 90+ ✓
- **不支持 IE 浏览器**

**Q：在手机上可以使用系统吗？**
A：
1. 系统界面支持手机浏览，但体验欠佳
2. 不建议在手机上进行考试和作业
3. 建议使用PC/Mac进行重要操作
4. 申请时如有原生应用计划，届时会单独推出

**Q：页面加载很慢怎么办？**
A：
1. 清除浏览器缓存：Ctrl+Shift+Delete（Windows）或Cmd+Shift+Delete（Mac）
2. 关闭浏览器标签页和其他应用，释放内存
3. 刷新页面或等待稍后再访问
4. 更换网络（如从WiFi切换到4G）

## 故障排除

### 8.1 常见错误与解决方案

**错误：用户名或密码错误**
```
现象：登录时提示"用户名或密码错误"
原因：
  1. 输入的用户ID或密码有误
  2. Caps Lock(大小写)被激活
  3. 账户被禁用或删除

解决：
  1. 检查用户ID是否正确(学号或工号)
  2. 关闭Caps Lock，重新输入密码
  3. 点击"忘记密码"重置
  4. 联系管理员检查账户状态
```

**错误：试卷提交失败**
```
现象：点击"提交试卷"后没有反应或报错
原因：
  1. 网络不稳定
  2. 浏览器问题
  3. 答案数据过大

解决：
  1. 检查网络连接，稍后重试
  2. 清除浏览器缓存，换浏览器尝试
  3. 如仍然失败，联系技术支持（告知时间和考试名称）
  注意：已保存的答案不会丢失
```

**错误：文件上传失败**
```
现象：上传图片、视频等文件时失败
原因：
  1. 文件超过大小限制(10MB)
  2. 文件格式不支持
  3. 网络上传中断

解决：
  1. 压缩文件到10MB以内
  2. 确认文件格式为系统支持的类型
  3. 重新上传
  4. 可分多次上传较大文件
```

### 8.2 性能优化建议

**网络方面：**
- 使用有线网络而非WiFi，连接更稳定
- 避免下载其他文件时进行考试
- 考试前关闭其他消耗带宽的应用

**浏览器方面：**
- 定期清除缓存（每月一次）
- 关闭不需要的浏览器标签页
- 升级浏览器到最新版本
- 减少已安装的扩展插件数量

**系统方面：**
- 重启电脑，清理系统临时文件
- 关闭杀毒软件的网络监控（如需要）
- 确保系统有足够的磁盘空间(>1GB)
